How to Compile: 

    To compile the file from a terminal simply navigate to the directory the file is in and type: g++ -o output-program Ex1.cpp

    Then to run the program, while in the same directory, type: ./output-program


How to run: 

    The program will ask you to enter the first number, simply type a single number and press enter, the program will then ask for a second number, type another number followed by enter.

    The output will show you the sum and product of the two entered numbers.
